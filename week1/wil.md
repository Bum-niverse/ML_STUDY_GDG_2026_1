## 📌 2. KNN 모델의 주요 포인트

KNN모델의 경우, 거리 기반 알고리즘이기 때문에 각 특성의 스케일에 큰 영향을 받는다.

해당 모델은 범위 값이 더 큰 것에 영향을 많이 받기 때문에, 데이터 표준화를 하지 않으면 오류가 날 수 있다.

새로운 데이터와 기존 데이터 간의 거리를 계산하여 가장 가까운 K개의 이웃을 선택, 다수결로 분류를 수행한다.

## 📌 3. 데이터 표준화 전과 후

데이터 표준화는 각 데이터에서 평균을 빼고, 표준편차로 나누어 평균을 0, 표준편차를 1로 설정하는 과정이다.

KNN의 경우 거리를 계산하는데, 해당 실습 간 길이는 [10, 40], 무게는 [10, 1000]의 범위로 무게의 영향을 많이 받을 수 밖에 없다.

표준화 전 작성한 코드의 경우 아래와 같다.

🔹 표준화 전 코드
---
```python
kn.fit(fish_data, fish_target)
pred = kn.predict(new_fish)
```

해당 코드를 작성했을 경우 새로운 데이터인 [25, 150]의 결과 값이 빙어로 나오게 된다.

무게의 영향을 많이 받았기 때문이다.

표준화를 위해 내가 작성한 코드의 경우 아래와 같다.

🔹 표준화 후 코드
---
```python
from sklearn.preprocessing import StandardScaler  # 데이터 표준화를 위한 클래스 불러오기

scaler = StandardScaler()  # StandardScaler 객체 생성 (표준화 도구 준비)

fish_data_scaled = scaler.fit_transform(fish_data)

# 학습 데이터의 평균과 표준편차를 계산(fit)한 후
# (x - 평균) / 표준편차 공식을 적용하여 데이터를 표준화(transform)

new_fish_scaled = scaler.transform(new_fish)

# 앞에서 구한 평균과 표준편차를 사용하여 새로운 데이터를 같은 기준으로 표준화
# (새 데이터에는 fit을 하면 안 되고 transform만 사용해야 함)

kn.fit(fish_data_scaled, fish_target)

# 표준화된 학습 데이터를 이용해 KNN 모델 학습

pred = kn.predict(new_fish_scaled)

# 표준화된 새로운 데이터를 입력으로 받아 도미/빙어 여부 예측
```
🔹 개념 정리
---
여기서 scaler는 표준화 도구 (계산기)이다.
fit()은 평균 및 표준 편차 계산 (계산기 설정),
transform()은 표준화 수행 (실행 함수),
fit_transform()은 fit과 transform을 한번에 수행 한다.

만약 기존 데이터가 아닌 새 데이터에 fit을 했을 경우 기존 데이터가 깨지기 때문에 이미 학습된 데이터를 기준으로 거리 계산을 하기 위해 기존 데이터로 표준화를 진행한다.

## 📌 4. 결과 및 비교

표준화 전에는 빙어로, 표준화 후에는 도미로 결과가 나온다.

표준화 전과 후로 KNN 시각화 그래프가 변하진 않는다.

시각화 그래프 상에는 도미 둘, 빙어 하나로 뜸에도 불구하고 표준화 전에는
[12, 10] -> 빙어 / [25, 150] -> 빙어로 결과 값이 나온다.

표준화 후에는
[12, 10] -> 빙어 / [25, 150] -> 도미로,
그래프 상으로 보이는 것과 동일하게 결과값이 나오는 것을 볼 수 있다.
